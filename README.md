# Promo Hub

Live: https://promohub.netlify.app/

## Description

Corporate website built with Next.js (App Router) including a custom contact API with reCAPTCHA validation and server-side email handling.
The project focuses on performance optimization, modular component architecture, and production-ready deployment.

## Stack
- Next.js (App Router)
- React
- REST API (custom API route)
- Nodemailer (email handling)
- reCAPTCHA v2 verification
- SCSS (modular styling)
- Docker (production setup)
- Netlify (live demo)

## Features
- Custom API endpoint (/api/contact)
- Server-side email sending 
- reCAPTCHA validation 
- Modular architecture 
- Lazy loading 
- Code splitting 
- Environment variable configuration 
- Production-ready build


## Local Development

First, run the development server:

```bash
npm install
# and
npm run dev
```
App runs on:
http://localhost:3000
